<h1>
  🏓 Ping Pong Scoreboard
</h1>

Marcador de ping pong interactivo y adaptable, diseñado para llevar el control de una partida de forma rápida, clara y cómoda desde cualquier dispositivo.

Permite configurar los puntos necesarios para ganar, el cambio de saque y un modo especial de saque de rescate. También incluye selección del primer jugador en sacar, intercambio de lados, control manual de la puntuación y modo pantalla completa.

Prepárate para **configurar**, **anotar** y **controlar** cada partido de forma sencilla.

✨ **Ping Pong Scoreboard** está desarrollado con **HTML, CSS y JavaScript**, con un diseño responsive pensado para funcionar correctamente tanto en ordenadores como en tablets y teléfonos móviles.

## 🕹️ Cómo utilizarlo

- **Nombres de los jugadores**: pulsa sobre el nombre para editarlo.
- **Puntos para ganar**: selecciona una partida a 11 o 21 puntos.
- **Cambio de saque**: elige si el saque cambia cada 2 o cada 5 puntos.
- **Saque de rescate**: activa esta opción para que, en la fase final, saque el jugador que va perdiendo.
- **Comenzar**: reinicia la puntuación y prepara una nueva partida.
- **Primer saque**: después de comenzar, pulsa sobre el jugador que realizará el primer saque.
- **Sumar puntos**: pulsa sobre la zona de un jugador o utiliza el botón `+`.
- **Restar puntos**: utiliza el botón `−`.
- **Cambiar lados**: pulsa el botón `⇄` para intercambiar jugadores, puntuaciones y colores.
- **Pantalla completa**: pulsa el icono de la esquina inferior derecha.

---

## 🏆 Reglas del marcador

- La partida termina cuando un jugador alcanza los puntos configurados.
- Para ganar debe existir una diferencia mínima de 2 puntos.
- El saque cambia automáticamente según la opción seleccionada.
- En la fase final de empate, el saque cambia después de cada punto.
- Al finalizar la partida se muestra el nombre del jugador ganador.

---

## 📱 Diseño responsive

El marcador se adapta automáticamente a diferentes tamaños y orientaciones:

- **Ordenador**: menú lateral y jugadores distribuidos horizontalmente.
- **Pantallas estrechas**: el menú se coloca encima del marcador.
- **Móvil o tablet vertical**: los controles se reorganizan para facilitar su uso táctil.
- **Móvil o tablet horizontal**: el marcador aprovecha toda la pantalla.
